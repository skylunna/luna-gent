# 🌙 Luna-Gent

A high-performance agent service based on Go and Rust

**高性能私有化 RAG Agent 引擎**
轻量｜安全｜无依赖｜可私有化部署

## 介绍

Luna-Gent 是一个基于 **Go + Rust** 构建的企业级文档问答 Agent。

- 本地解析文档
- 本地向量检索
- 兼容所有大厂模型 API
- 一键编译，无环境依赖

## 架构

- **go/**：服务层、CLI、API、模型调用
- **rust/**：文档解析、向量计算、高性能引擎

## 技术栈

- Go：服务化、高并发、部署简单
- Rust：极致性能、内存安全

## 状态

🚧 开发中

## 开源目标

打造最容易部署、最轻量、最安全的本地文档问答 Agent。
